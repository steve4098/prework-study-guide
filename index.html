HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML forms the first outline shape of the house

CSS

  • .card:assigns the rules to all elements with this class
  • colours can be edited with the visual block or using HEX code
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.
  • CSS gives the house its colours and details

Git

  • git status: checks which branch we are on
  • git checkout -b branch-name: creates a new branch with this name
  • git checkout 'name':moves to that branch or main
  • git pull origin main:pulls down most up to date version of main
  • code .:launches VSC
  • cd bootcamp:moves directory to bootcamp folder
  • ls: lists all files in current folder
  • git add -A: lines up the changes to be commited
  • git commit -m "what I've done": commits the changes to GitHub to be reviewed

JavaScript

  • WAY more complex!
  • functions are a group of instructions - each instruction lists as a console.log
  • A variable is a named container that allows us to store data in our code.
  • in an array, the items are indexed (given a number) and the first item is always 0 (not 1)
  • Control flow is the order in which a computer executes code in a file - usually top down
  • if statements are litereally yes/no gates to pass through to determine a response
  • === forces a true/false boolean response
